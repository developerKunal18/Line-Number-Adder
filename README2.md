# Line Number Adder

A Python script that adds line numbers to multiple lines of text.

## Description
This project accepts multiple lines of input and prints each line with an automatic line number.

## Features
- Multiple line input
- Automatic numbering
- Simple text formatting
- Beginner-friendly

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python line_number_adder.py

4. Enter lines of text
5. Type `done` to finish

## Example

Input:
Python
Java
C++
done

Output:
1. Python
2. Java
3. C++
